## Como publicar

1. Coloque um logo em `assets/logo.png` (opcional, mas recomendado).
2. Pode substituir o favicon em `assets/favicon.ico`.
3. Faça upload da pasta `angola-verde` para um servidor web estático:
   - [Netlify](https://netlify.com) (arrastar pasta)
   - [Vercel](https://vercel.com)
   - [GitHub Pages](https://pages.github.com)
